Python Keylogger

This repository contains a minimal keylogger written in Python using the pynput library. It captures keyboard input and logs it to a local file (key_log.txt). This tool is intended strictly for educational, ethical, and consent-based use cases such as:

Demonstrating how keylogging works
Teaching secure coding practices
Exploring input event handling in Python
Ethical Disclaimer

This project is for learning purposes only. Do not use it to monitor or record input without explicit consent. Unauthorized use may violate privacy laws and ethical standards.

Requirements

Python 3.x
pynput library
Install dependencies:

bash pip install pynput
